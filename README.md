Next Base Auth
This project is built with Next.js, TypeScript, Tailwind CSS, and Shadcn UI.

## Technologies
- **Framework:** Next.js
- **Language:** TypeScript
- **Styling:** Tailwind CSS, Shadcn UI
- **Runtime:** Node.js

## Getting Started

To install dependencies:
```
npm run install
```

To run:
```
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) in your browser.
